## Project Scope

This project analyzes the sentiment of user-entered text using Python and TextBlob. It classifies text as positive, negative, or neutral.

## Features

- User text input
- Sentiment prediction
- Positive sentiment detection
- Negative sentiment detection
- Neutral sentiment detection

## Technologies Used

- Python
- TextBlob
- Pandas
